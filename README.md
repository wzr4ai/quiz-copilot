# Quiz Copilot 使用指南

Quiz Copilot 提供题库练习、录题、批量导入与管理功能，支持 AI 辅助导题与解析。前端基于 UniApp，后端基于 FastAPI。

## 1. 体验入口
- Web/H5（需自行部署后端与前端）：
  - 后端启动后，将前端 `uni-app/` 的 `API_BASE` 配置指向后端地址，再运行/打包 UniApp。
- 登录 / 注册：在“我的”页面完成。登录后可练习、收藏；管理员拥有题库管理、质检入口。

## 2. 主要功能
- 练习与答题：按题库练习，支持收藏与错题记录。
- 录题与导入：管理员可在“题库管理”中新建题库、录入题目，或批量导入处理后的题库文件。
- AI 辅助：
  - 文本/图片导题（ZAI/Gemini 接入）。
  - 自动补全/质检解析（glm-4.5），疑似错题记录。
- 快速定位与修正（管理员）：
  - 按题目 ID 跳转编辑。
  - 关键词搜索题干/答案。
  - “疑似错题”列表查看日志、编辑题目并标记状态。

## 3. 管理员常见操作
1) 批量导入：在“题库管理”填写服务器目录，或先用 `question_bank_tool.py` 分割生成 `.converted.json/.chunks.jsonl` 再导入。  
2) QID 编辑：输入题目 ID 跳转到独立编辑页。  
3) 疑似错题：从日志导入 DB 后，可在“疑似错题”页面按列表选择、查看 reason、编辑并标记状态。  
4) 搜索：按关键字检索题干/答案，查看匹配题并跳转编辑。

## 4. 账号与安全
- JWT 登录，默认有效期 60 分钟（管理员可在后端 `.env` 调整 `ACCESS_TOKEN_EXPIRE_MINUTES`）。
- 管理员功能仅 admin 可见/可用；普通用户仅可练习与收藏。
